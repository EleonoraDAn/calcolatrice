# Calcolatrice Web

Una semplice calcolatrice web che permette di eseguire operazioni matematiche di base.

## Descrizione

Questo progetto è una calcolatrice interattiva sviluppata utilizzando HTML, CSS e JavaScript. L'interfaccia utente è composta da un display per mostrare i numeri e i risultati, e da pulsanti per inserire numeri e operazioni.

## Funzionalità

- Operazioni aritmetiche di base: addizione (+), sottrazione (-), moltiplicazione (x), divisione (/), modulo (%)
- Calcolo della radice quadrata (√)
- Pulsanti per cancellare l'ultimo carattere (C) e resettare completamente (CANC)
- Display di input readonly per mostrare l'espressione e il risultato

## Come utilizzare

1. Apri il file `index.html` in un browser web moderno.
2. Clicca sui pulsanti numerici per inserire i numeri.
3. Clicca sui pulsanti delle operazioni per selezionare l'operazione desiderata.
4. Premi il pulsante "=" per calcolare il risultato.
5. Usa "C" per cancellare l'ultimo carattere o "CANC" per resettare tutto.

## Struttura del progetto

- `index.html`: Il file principale HTML che contiene la struttura della pagina e i pulsanti.
- `css/calcolatrice.css`: Il foglio di stile per l'aspetto visivo della calcolatrice.
- `javascript/calcolatrice.js`: Il codice JavaScript che gestisce la logica delle operazioni.
- `img/`: Cartella contenente le immagini di sfondo utilizzate nel design.

## Tecnologie utilizzate

- HTML5
- CSS3
- JavaScript (ES6+)

## Requisiti

- Un browser web moderno che supporti HTML5, CSS3 e JavaScript.
